Registered 2006-03-07 by HP Linux Imaging and Printing

HPLIP is an HP developed solution for printing, scanning, and faxing with HP inkjet and laser based printers in Linux.

The HPLIP project provides printing support for over 2,080 printer models--including; Deskjet, Officejet, Photosmart, PSC (Print, Scan, Copy), Business Inkjet, LaserJet, Edgeline MFP, and LaserJet MFP.

Latest HPLIP Release is available to download from https://developers.hp.com/hp-linux-imaging-and-printing

Release notes is available from https://developers.hp.com/hp-linux-imaging-and-printing/release_notes

Note: HPLIP is free, open source software distributed under the MIT, BSD, and GPL licenses.

Note: Most HP models are supported, but a few are not. See Supported Devices at the HPLIP website for more information:

https://developers.hp.com/hp-linux-imaging-and-printing/supported_devices/index

Note: Before submitting tickets for support please check the "Browse Knowledge Base" section from (https://developers.hp.com/hp-linux-imaging-and-printing) for possible answers to your questions/or for troubleshooting your problem.

Note: If you are experiencing a hardware problem with your HP printer (or) Windows and OS X users seeking drivers, please contact HP Support. (http://hp.com/support).
HP LIP Software does NOT support either of Windows / OS X operating systems

Note: All support is provided is online and only by clicking the "Ask a Question" or "Report a bug" button below.

PLEASE READ To make sure you are submitting your problem to the correct area, use this guideline:
1. FOR END USER SUPPORT: If you are a end user and having a problem, no matter the problem, Please use the "Ask a question" button below. MOST USERS DO THIS.
2. DEVELOPERS: If you are a developer or distribution representative and have a defect to report please submit under "Report a bug". Most users do NOT use this feature.
